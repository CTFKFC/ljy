《越集》后续新增 / 删除文章操作说明

一、这次改了什么
1. 首页删掉了偏解释性、重复性的长段废话
2. 保留原来“烬与月”的气质、配色和氛围
3. 完整正文改到 poems/ 目录下的独立单篇页
4. 首页只保留：故事脉络、目录、少量摘读
5. 首页目录会根据 content.js 自动生成

二、以后新增文章
1. 打开 content.js
2. 在 poems 数组末尾新增一项，例如：

{
  "title": "新文章标题",
  "slug": "34-新文章标题",
  "content": "第一段\n\n第二段"
}

3. 复制 poems/ 里任意一个 html 文件作为模板
4. 把文件名改成：34-新文章标题.html
5. 把页面里的标题、正文、上一篇/下一篇链接改掉
6. 如果你希望它也出现在首页摘读区，就把标题加入 featured_titles 数组
7. 上传覆盖原站文件即可

三、以后删除文章
1. 打开 content.js
2. 删除 poems 数组里对应文章对象
3. 删除 poems/ 目录里对应 html 文件
4. 如果 featured_titles 里有它，也删掉
5. 如果它前后两篇还有上一篇/下一篇按钮，记得顺手改一下链接
6. 上传覆盖原站文件即可

四、最重要的维护规则
- 文章目录以 content.js 为准
- 单篇正文页以 poems/ 目录文件为准
- 新增文章时，content.js 和 poems/ 对应单篇页要一起加
- 删除文章时，两边也要一起删

五、如果你想以后更省事
后面我可以再帮你改成“只改 content.js，就能自动批量生成全部单篇页”的版本。
这样以后新增/删除文章就不用再手工改 poems/ 目录里的 html 文件了。
